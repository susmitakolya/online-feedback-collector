# online-feedback-collector

(i) Description

A Flask-based web application to collect user feedback and display results in an admin dashboard.

(ii) Technologies Used
HTML, CSS, JavaScript
Python (Flask)
SQLite
Chart.js

(iii) How to Run
Install Flask
Run python app.py
Open browser at http://127.0.0.1:5001/

(iv) Features
Feedback submission
SQLite database
Admin dashboard
Data visualization
